# yggycs-Blog
## 项目介绍
本项目是为了走一遍前后端流程，目的是开发一个属于自己的个人网站，可以让自己在上面发一些博客啥的。一方面可以锻炼一下自己的动手能力，另一方面防止自己毕业了啥都干不了学习点技能防止自己饿死，还有就是可以把一些容易忘的东西记在这个网站上，当自己的技能备忘录。
